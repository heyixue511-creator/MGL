# 设计的“跨边界”_赵伟 摘要版

## 1. 文献定位

《设计的“跨边界”——论维克多·马格林的设计批评思想》是一篇以马格林设计批评思想为研究对象的硕士学位论文。论文把“跨边界”作为贯穿马格林设计思想的一条主线，试图从设计学边界、设计平台边界、设计研究边界三个层面解释马格林思想的开放性和扩展性。

## 2. 核心问题

论文关注的问题主要包括：

1. 马格林如何理解设计边界的模糊性与扩展性；
2. 他如何看待设计师在团队、社会和跨学科实践中的多重角色；
3. 他如何理解设计史与设计研究之争；
4. 他如何通过“社会模型”突破市场导向设计；
5. 他如何把食品研究、博物馆收藏、消费文化、产品经验、产品环境和设计伦理纳入设计研究视野。

## 3. 章节结构概括

论文大体分为五个部分：

- 第一章讨论“跨边界”设计思想产生的背景，包括学科间性、非物质社会、设计方法运动和设计科学等；
- 第二章分析马格林关于设计学边界的思想，涉及设计定义、设计师角色、设计教育和设计史研究；
- 第三章讨论设计学平台的跨边界，尤其是设计研究社区与社会模型；
- 第四章讨论设计研究的跨边界，包括设计与食品、消费文化、产品环境、设计伦理和未来构想；
- 第五章对马格林“跨边界”思想进行批评与意义评估。

## 4. 与马格林研究的关系

该论文的价值在于将马格林分散在设计史、设计研究、设计批评、社会模型和未来设计中的观点，归纳为“跨边界”思想。它强调马格林不是单纯的设计史家，而是通过设计史、设计研究、设计批评和设计教育共同推动设计学开放化的理论家。

## 5. 可用于论文写作的观点

可将本文作为中文研究现状中的一项代表性成果，用来说明国内学界已经开始从“设计批评思想”与“跨学科设计研究”的角度理解马格林。其可用观点包括：

- 马格林设计思想的核心不是封闭学科建设，而是边界扩展；
- 设计史与设计研究之争体现了设计学自我定位的焦虑；
- “社会模型”是马格林跨越市场设计边界的重要实践方案；
- 产品环境、消费文化和设计伦理构成其设计研究扩展的重要方向。
