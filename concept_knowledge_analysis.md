## 使用说明 (User Guide)

本文件是"从风格史到物质文化：设计史研究的方法论转向"知识体系的配套概念分析工具。收录维度一（认识论与本体论基础）下专题域1.1（物/客体问题域与物质性论争）的全部10个核心概念。

**条目结构**：每个概念包含英文术语、中文译名、精确定义、来源文献（作者、标题、年代）、学科来源（disciplinary origin），以及与其他概念的交叉引用关系。

**交叉引用**：概念之间的引用关系标记为"参见"，涵盖：(a) 概念性关联（共享理论前提或对立立场）；(b) 论证性关联（一个概念为另一个概念提供论证资源或批判对象）；(c) 来源性关联（来自同一作者/文献）。

**概念排序**：按英文术语字母顺序排列。

### C01: Corporeal Imagination / 身体性想象

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Corporeal Imagination |
| **中文译名** | 身体性想象 |
| **定义** | 每个社会通过对主体感官的特定"心智的社会化"（socialization of the psyche）将自身施加于主体的机制，物质性本身正是通过这种机制被感知的。对事物的感知不是自然的、普遍的，而是制度性地被塑造的——不同的社会组织模式产生不同的"物感"（sense of things）。 |
| **来源文献** | Bill Brown, "Thing Theory" (2001) |
| **理论渊源** | 概念借自希腊-法国哲学家 Cornelius Castoriadis（科内利乌斯·卡斯托里亚迪斯，1922-1997）的《社会的建制性想象》（*The Imaginary Institution of Society*, 1975）。卡斯托里亚迪斯论证每个社会通过"根本想象"（radical imaginary）创造自身独特的意义与制度世界。布朗将其转用于分析物质性感知的制度性条件。 |
| **学科来源** | 哲学（社会本体论）→ 批判理论 → 文学/文化研究 |
| **概念层级** | 元理论概念（meta-theoretical），为物质性效应概念提供社会-制度基础 |
| **参见** | → Materiality-Effect (C05)：物质性效应是身体性想象的具体运作机制；→ Object/Thing Distinction (C06)：两种不同的物感模式分别对应object（透过观看）和thing（被物质性打断）；→ Semantic Irreducibility of Things (C08)：物的语义溢出挑战既有分类体系，暗示身体性想象的可变边界 |

### C02: Flux of Materials / 材料的流变

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Flux of Materials |
| **中文译名** | 材料的流变 |
| **定义** | 材料不是等待文化铭刻的惰性底基（inert substrates），而是积极地处在运动之中——流动、混合、凝结、分散、蒸发和转化。"物"（things）只是这些持续的物质流中的临时凝结（temporary congealments）。材料始终处在生成（becoming）而非存在（being）的状态中。 |
| **来源文献** | Tim Ingold, "Materials against materiality" (2007) |
| **理论渊源** | 英戈尔德反对西方哲学传统中自亚里士多德以降将物质等同于惰性质料的倾向，转而汲取赫拉克利特（Heraclitus）的流变哲学、德勒兹（Deleuze）和瓜塔里（Guattari）的生成本体论，以及人类学中对"生命物质"（living matter）的认知传统。 |
| **学科来源** | 人类学（生态人类学、物质文化研究） |
| **概念层级** | 本体论概念，为材料vs.物质性区分提供物质世界的基本图景 |
| **参见** | → Materials vs. Materiality (C04)：流变本体论是对抽象物质性概念的根本替代；→ Properties as Histories (C07)：属性是流变中的事件而非固定的本质；→ Things Are in Life (C10)：物被卷入生命世界的流变中是其活跃性的来源；→ Hylomorphic Model Critique (C03)：形质论假设惰性质料，与流变本体论直接对立 |

---

### C03: Hylomorphic Model Critique / 形质论模型及其批判

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Hylomorphic Model Critique |
| **中文译名** | 形质论模型及其批判 |
| **定义** | 对亚里士多德"形质论"（hylomorphism）学说的批判：古典模型认为形式（morphe）是由能动的心灵（或设计者）施加在惰性质料（hyle）之上的，制造是被预想的形式在物质上的强制执行。英戈尔德论证此模型根本上歪曲了"制作"实际发生的方式——形式是在物质性参与的过程中从材料内部涌现出来的（emergent），而非被预想然后被强加的。形质论构成了物质文化研究和设计理论的未被检验的认识论预设。 |
| **来源文献** | Tim Ingold, "Materials against materiality" (2007) |
| **理论渊源** | 批判对象：亚里士多德《形而上学》和《物理学》中奠定的质料-形式二元论。英戈尔德同时也批判了这一模型在现代设计理论中的继承者——大卫·派伊（David Pye）对客观材料属性与主观质性的二元区分，认为这不过是形质论的现代版本。 |
| **学科来源** | 哲学（古希腊形而上学）→ 人类学（物质文化研究批判）→ 设计理论 |
| **概念层级** | 元理论批判概念，针对支配西方制造哲学两千余年的根本预设 |
| **参见** | → Materials vs. Materiality (C04)：物质性概念即为形质论中"hyle"（质料）的现代抽象化；→ Flux of Materials (C02)：流变本体论是形质论的直接对立面；→ Properties as Histories (C07)：过程性属性观是对形质论的替换方案；→ Materials vs. Materiality (C04)：英戈尔德指出派伊（Pye）的材料-属性区分只是形质论的翻版 |

### C04: Materials vs. Materiality / 材料 vs. 物质性

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Materials vs. Materiality |
| **中文译名** | 材料 vs. 物质性 |
| **定义** | 材料（materials）是事物实际由构成的实在物质（石头、木材、黏土、金属、纤维），具有可触、可感、可操作的物理属性；物质性（materiality）则是抽象的哲学概念，指涉事物的"物般的"品质（thingly quality），是理论反思的对象而非实践参与的对象。英戈尔德的核心论证：物质文化研究的文献过度迷恋"物质性"的抽象讨论，却系统性地忽略了对实在材料的实践性接触和知识。 |
| **来源文献** | Tim Ingold, "Materials against materiality" (2007) |
| **理论渊源** | 英戈尔德将物质文化研究（material culture studies）在人类学和考古学中的整体传统作为批判对象——包括丹尼尔·米勒（Daniel Miller）的物质文化理论和皮埃尔·布尔迪厄（Pierre Bourdieu）的实践理论中对"物"的讨论——认为它们均陷入了从具体材料到抽象物质性的概念升华。他也批判了大卫·派伊（David Pye）对客观属性与主观质性的区分。 |
| **学科来源** | 人类学（生态人类学、物质文化研究） |
| **概念层级** | 本体论-方法论区分概念，是英戈尔德全部论证的基础性对立 |
| **参见** | → Flux of Materials (C02)：材料的流变本是其本质特征，物质性概念则遮蔽了这种动态性；→ Properties as Histories (C07)：属性属于材料（过程性的），而非物质性（抽象化的）；→ Hylomorphic Model Critique (C03)：物质性概念是形质论中hyle的现代继承；→ Things Are in Life (C10)：只有材料在生命之流中，物质性作为抽象概念不在 |

### C05: Materiality-Effect / 物质性效应

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Materiality-Effect |
| **中文译名** | 物质性效应 |
| **定义** | 物质性不是粗暴的给定（brute given），而是通过再现（representation）、感知（perception）和社会建制（social institution）产生出来的*效应*。某些再现的片段被赋予了实在之索引的重量（the weight of an index of reality），并被稳定化为对事物的感知——仿佛事物本身就"拥有"物质性一般。然而这种稳定化从来不是一劳永逸的；物质性总是在被生产和再生产的半稳定状态中。 |
| **来源文献** | Bill Brown, "Thing Theory" (2001) |
| **理论渊源** | 布朗此概念深受后结构主义和话语理论的影响——尤其是罗兰·巴特（Roland Barthes）的"真实效应"（effet de reel）概念和米歇尔·福柯（Michel Foucault）对话语/物质性关系的分析。但布朗将此逻辑从文本再现扩展至物质文化领域。该概念也与卡斯托里亚迪斯的身体性想象概念内在相关。 |
| **学科来源** | 批判理论 → 文学/文化研究 → 物质文化研究 |
| **概念层级** | 认识论概念，解释物质性感知的社会生产机制 |
| **参见** | → Corporeal Imagination (C01)：身体性想象是物质性效应得以运作的制度性条件；→ Object/Thing Distinction (C06)：物质性效应是object被感知和stabilized为"物"的过程；→ Thingness as Latency and Excess (C09)：物质性效应试图驯化thingness的溢出，但永远不能完全成功 |

### C06: Object/Thing Distinction / 物/客体区分

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Object/Thing Distinction |
| **中文译名** | 物/客体区分 |
| **定义** | 物（object）是我们透过它来看的东西——通向意义、历史、社会或文化的透明通道；东西/客体（thing）在物不再为我们运作时方才自我呈现——当物的物质性中断了其符号性透明，我们才瞥见东西。事物性（thingness）指涉的是一种特定的主客体*关系*，而非物的固有本体属性。核心命题：我们透过物来看（look through objects），但对东西只能瞥见一斑（catch only glimpses of things）。 |
| **来源文献** | Bill Brown, "Thing Theory" (2001) |
| **理论渊源** | 布朗从海德格尔（Martin Heidegger）的"上手之物"（Zuhandenheit / ready-to-hand）与"在手之物"（Vorhandenheit / present-at-hand）的区分中获得灵感，但将其从存在论哲学转向文化分析。海德格尔的锤子只有在坏了的时候才从透明的工具变为引人注目的"东西"，布朗将此存在论洞见扩展为分析文化、文学和物质生活的批判框架。 |
| **学科来源** | 现象学（海德格尔）→ 文学/文化研究 → 批判理论 → 物质文化研究 |
| **概念层级** | 元理论-本体论概念，是整个"物理论"框架的核心区分 |
| **参见** | → Thingness as Latency and Excess (C09)：对thingness更详细的时间性分析；→ Materiality-Effect (C05)：解释了object如何被感知为"物"的稳定化过程；→ Semantic Irreducibility of Things (C08)：thing一词的语义歧义性映射了object/thing的概念区分；→ Corporeal Imagination (C01)：不同的社会制度生产不同的object/thing感知模式 |

---

### C07: Properties as Histories / 属性作为历史

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Properties as Histories (Processual and Relational) |
| **中文译名** | 属性作为历史（过程性与关系性） |
| **定义** | 材料的属性不是物质的固定的、本质的、内在的属性，而是过程性的（processual）、关系性的（relational）、涌现的（emergent）——它们*发生*（occur）而非*存在*（exist）。每一种属性都是一个浓缩的故事（condensed story），讲述着材料在流动、混合、变异、与环境互动中所经历的一切。例如，木头的"硬度"不是一个固定的数值，而是该木材的生长史、砍伐后的干燥史、与环境湿度交互史的凝结。 |
| **来源文献** | Tim Ingold, "Materials against materiality" (2007) |
| **理论渊源** | 英戈尔德将怀特海（A. N. Whitehead）的过程哲学和德勒兹（Deleuze）的差异本体论与人类学田野经验相结合，论证了西方科学中将材料属性还原为可测量的、非时间性的物理参数的局限性。同时批判了大卫·派伊（David Pye）对客观材料属性与主观质性的二元区分。 |
| **学科来源** | 人类学（生态人类学）→ 过程哲学 → 材料科学批判 |
| **概念层级** | 本体论-认识论概念，为"材料vs.物质性"区分提供更具体的属性理论 |
| **参见** | → Materials vs. Materiality (C04)：属性属于材料（动态的），物质性概念则将属性抽象化为非时间的；→ Flux of Materials (C02)：属性是材料流变史中的事件凝结；→ Hylomorphic Model Critique (C03)：形质论预设了材料的固定属性等待形式施加；→ Things Are in Life (C10)：物在生命世界中，其属性随生命世界的变迁而变迁 |

### C08: Semantic Irreducibility of Things / 物的语义不可还原性

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Semantic Irreducibility of Things |
| **中文译名** | 物的语义不可还原性 |
| **定义** | 英语词"thing"承载着独特的、不可还原的歧义性：它既指涉巨大的总体性（"the thing is the sum of the world"），又指涉具体的个别性（"that green thing in the hall"）；它既可以是法律-政治事件（古英语和中古英语中的þing——集会、法庭、争论），又可以是物理实体。词"thing"指向一个边界或阈限（limit or liminality），悬浮在可命名与不可命名之间。这种语义不可还原性不是语言学的缺陷，而是理论资源——它指示着物质文化超出我们分类和指意体系的那个界限本身。 |
| **来源文献** | Bill Brown, "Thing Theory" (2001) |
| **理论渊源** | 布朗从海德格尔对"物"（das Ding）的词源学分析中获得灵感，兼取英古英语和中古英语中þing（集会、法庭、公共事务）的语义史——暗示"物"从一开始就不是纯粹的物质实体，而是社会-法律-政治关系的凝结。布朗将此语义学洞见与后结构主义对指意过程（signification）的分析相结合。 |
| **学科来源** | 语言-语义学 → 现象学（海德格尔）→ 文学/文化研究 |
| **概念层级** | 语义-认识论概念，为整个"物理论"提供语言哲学基础 |
| **参见** | → Object/Thing Distinction (C06)：物的语义歧义性映射到object/thing的概念区分上；→ Thingness as Latency and Excess (C09)：语义溢出是物性（thingness）的语言维度的表现；→ The Corporeal Imagination (C01)：物的可命名/不可命名的边界是制度性地被生产的 |

### C09: Thingness as Latency and Excess / 物性作为潜在性与溢出

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Thingness as Latency and Excess |
| **中文译名** | 物性作为潜在性与溢出 |
| **定义** | 物性（thingness）在时间上同时以两种模态运作：(1) 作为"之前"（the before）——物（object）从中被物质化出来的无定形的物理性，即材料在获得可辨识的文化形式之前的那个"物本身"；(2) 作为"之后"（the after）——物中在物理上或形而上学上不可还原为"物"的残余，即物的感性的或形而上学层面的在场（presence），那种使物成为价值、恋物、偶像和图腾的"魔力"（magic）。关键论点：object/thing的辩证法不是时序性的（先有前者后有后者），而是*同时性的*（simultaneous）——每一个物都同时既是"透明通道"又是"不透明在场"。 |
| **来源文献** | Bill Brown, "Thing Theory" (2001) |
| **理论渊源** | 布朗综合了马克思主义的商品拜物教理论（马克思的"商品的形而上学微妙和神学怪诞"）、弗洛伊德的精神分析中关于恋物（fetish）的讨论，以及人类学中关于"物灵"（mana）和图腾的经典研究。潜在性（latency）概念受弗洛伊德的影响，溢出（excess）概念则来自巴塔耶（Bataille）和一般的超量经济学思想。 |
| **学科来源** | 马克思主义 → 精神分析 → 人类学 → 文学/文化研究 |
| **概念层级** | 本体论-时间性概念，是object/thing区分的深层时间结构分析 |
| **参见** | → Object/Thing Distinction (C06)：对此概念的更详细的时间性展开；→ Semantic Irreducibility of Things (C08)：语义不可还原性是物性溢出的语言维度的表现；→ Materiality-Effect (C05)：物质性效应是驯化物性溢出（将thing转化为可读的object）的社会机制；→ Corporeal Imagination (C01)：不同的物感制度产生不同的物性溢出体验 |

### C10: Things Are in Life / 物在生命中

| 字段 | 内容 |
|:---|:---|
| **英文术语** | Things Are in Life (vs. Life Is in Things) |
| **中文译名** | 物在生命中（而非生命在物中） |
| **定义** | 英戈尔德颠倒了人类学和物质文化研究中常见的泛灵论假设（animistic assumption）——即认为物之所以"有生命"是因为它们被赋予了能动性、精神或主体性。英戈尔德的替代性命题是：物之所以是活跃的，不是因为生命被注入了物之中，而是因为物被卷入了维系它们的、生活世界的生成性流变和循环性流动之中。物在生命的流动中"活着"，如同漩涡在水流中"活着"一样——它们的活性来自其所参与的流动，而非来自其"内部"的某种本质。 |
| **来源文献** | Tim Ingold, "Materials against materiality" (2007) |
| **理论渊源** | 英戈尔德直接批判了物质文化研究中将"能动性"赋予物的倾向（包括拉图尔的ANT和盖尔的艺术能动性理论），认为这不过是将西方的主体概念投射到了物上。他的替代方案汲取了怀特海（Whitehead）的过程哲学、梅洛-庞蒂（Merleau-Ponty）的身体现象学以及他与原住民社群长期田野工作所获得的非西方本体论知识。 |
| **学科来源** | 人类学（生态人类学）→ 过程哲学 → 现象学 |
| **概念层级** | 本体论概念，为理解物的"活跃性"提供非泛灵论的替代方案 |
